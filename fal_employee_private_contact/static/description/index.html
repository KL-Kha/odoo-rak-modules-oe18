Story behind the features

Odoo behavior : Employee private address is visible for any Odoo user (public.)

CLuedoo behavior : Cluedoo introduces the notion of private address and force the employee private address to be of type private. Then only user with this access right level can see it.

Module + : It's a legal requirement to keep the private address of employees confidential. Just a must.

How to Use this Module?

Access to Private Address

On user access right, we have Access to Private Addresses.
Private Contacts

User with Access to Private Address can access Private Contacts menu.
Private Information

On Employee form view, user with Access to Private Address can access Private Information tab and they can fill private information of the employee.
