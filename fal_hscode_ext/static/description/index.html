Story behind the features

Odoo behavior : Odoo proposes HS Code only at product level.

Cluedoo behavior : As for all the fields which could have the same property for a whole category, Cluedoo add the option to configure HS Code on the product category directly. By just configuring the product category, you will get clean invoice and deliveries documents including HS Code.

Module + : Put configuration at the product category level to simplify the configuration management.

CLuedoo Recomendation : To be installed on 100% of the database using HS Code notion.
