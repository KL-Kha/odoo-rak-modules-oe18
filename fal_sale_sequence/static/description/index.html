Quotation Sequence

Split the quotation sequence and the order sequence in sales.

CLuedoo

Story behind the features

Odoo behavior : Odoo use the same sequence both for quotation and order.

CLuedoo behavior : Cluedoo gives one specific sequence for quotation and for order. So when you confirm a quotation, you will get an unique order number.

Module + : From a business and legal point of view, quotation and order are completely different, even if the 2nd is the continuity of the first one. Indeed, a quotation is a way to communicate with 3rd party, without any other value than communication when an order is equivalent to a contract with a strong legal value. So it's important to have a special marker to differentiate these 2 kinds of documents and the sequence is a simple and efficient tool.

CLuedoo recommendation : To be installed on 100% of the database. On each database we did not install this module at the beginning because it does not seem required, we finally decided to install after a few years to clarify the border between order and quotation. Related modules: Quotation versioning.

Our Feature as Solution

Add quotation number sequence in sales order

How to Use this Module?

Go to Sales > Orders > Quotation. Create new Quotation and it will automatically create the quotation number.
